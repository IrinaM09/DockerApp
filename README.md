# DockerApp
This repo is used together with DockerHub to automatically build a Weather Application.\
You can enter as a guest or sign up for an account in which case you can save and delete your favourite locations.

### Arhitecture ### 
- Angular
- Java
- Python
- Elasticsearch
- Grafana
